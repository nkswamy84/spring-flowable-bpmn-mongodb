# Flowable Tutorial

## Overview
This repository provides a tutorial for **Flowable**, a lightweight and powerful **Business Process Management (BPM)** engine. It covers key concepts, including **process modeling**, **execution**, and **integration** with Java and Spring Boot.

## Features
- Hands-on examples for **Flowable BPM**
- Integration with **Spring Boot**
- Demonstration of **Process, Task, and Case Management**
- Event-driven workflows using **Kafka**
- REST API implementation for process execution

## Prerequisites
Before running this project, ensure you have:
- **Java 17+**
- **Maven**
- **Docker (Optional, for database setup)**
- **Spring Boot dependencies** installed
